# Coffee Leaf Disease Prediction Using CNN and VGG16

## Project Overview
This project uses Deep Learning techniques to identify coffee leaf diseases from images.

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Pandas
- CNN
- VGG16

## Features
- Image classification
- Transfer learning using VGG16,CNN
- Data preprocessing and augmentation
- Disease prediction from leaf images
